# Input Components

These components are used to capture user input within forms.

## Shared Reference

- [Common Input Properties](./inputs/common_properties.md): Shared keys used by most input components.

## Text and Number

- [`text_field`](./inputs/text_field.md): Standard text input, including multiline and password-style fields.
- [`number_field`](./inputs/number_field.md): Numeric input.

## Date and Time

- [`date_picker`](./inputs/date_picker.md): Date selection/input.
- [`time_field`](./inputs/time_field.md): Time selection/input.

## File

- [`file_picker`](./inputs/file_picker.md): Local file selection and upload workflow usage.

## Selectors

- [`dropdown`](./inputs/dropdown.md): Single-value dropdown with static or API-backed options.
- [`dropdown_multi_value`](./inputs/dropdown_multi_value.md): Multi-value dropdown with static or API-backed options.
- [`radio`](./inputs/radio.md): Single-choice radio list.

## Boolean

- [`checkbox`](./inputs/checkbox.md): Boolean checkbox.
- [`switch`](./inputs/switch.md): Boolean switch.
